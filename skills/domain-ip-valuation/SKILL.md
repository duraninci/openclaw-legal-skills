# Domain & IP Valuation Skill

You analyze domain names and intellectual property assets to assess fair market value, evaluate contribution amounts in business agreements, and identify valuation issues.

**Important**: Domain and IP valuation is inherently subjective. This provides analytical frameworks, not definitive valuations. Professional appraisers should be consulted for high-value transactions.

## Premium Domain Valuation Framework

### Domain Value Factors

**Tier 1: Intrinsic Factors**
| Factor | Impact | Premium Range |
|--------|--------|---------------|
| **Length** | Shorter = more valuable | 1-2 char: extreme premium; 3-4 char: high; 5-7: moderate |
| **Extension** | .com dominates | .com: 10-100x vs other TLDs |
| **Word type** | Dictionary > brandable > random | Dictionary: 10-50x premium |
| **Memorability** | Easy to spell/remember | 2-5x for highly memorable |
| **Pronunciation** | Easy to say | 1.5-3x for phonetically simple |
| **Typo risk** | Low confusion | -20-50% for easily mistyped |

**Tier 2: Commercial Factors**
| Factor | Impact | Premium Range |
|--------|--------|---------------|
| **Exact match keyword** | Matches search query | 2-10x for high-volume keywords |
| **Industry value** | Premium industries pay more | Finance, insurance, crypto: 5-20x |
| **Brandability** | Suitable for major brand | 2-10x for highly brandable |
| **Category ownership** | Defines a category | Extreme premium (100x+) |
| **Search volume** | Monthly searches | Higher volume = higher value |
| **CPC value** | Cost per click in industry | High CPC = higher value |

**Tier 3: Market Factors**
| Factor | Impact | Notes |
|--------|--------|-------|
| **Comparable sales** | Most important | Similar domains sold recently |
| **Current traffic** | Direct type-in value | Revenue multiple applies |
| **Existing revenue** | Parking/affiliate income | 2-5x annual revenue |
| **Development potential** | Business buildability | Future use cases |
| **Buyer urgency** | Negotiation leverage | Motivated buyer pays premium |

### Category-Defining Domain Premium

**What Makes a Category-Defining Domain?**
- Single generic word describing entire industry
- Would be the obvious choice for industry leader
- No modifier needed (not "best-X" or "X-online")
- Establishes instant authority and trust

### Valuation Methods

**Method 1: Comparable Sales Analysis**
```
1. Find 5-10 similar domain sales
2. Adjust for differences (length, extension, category)
3. Calculate adjusted average
4. Apply premium/discount for specific factors

Example:
- Shop.com sold for $3.5M
- Store.com sold for $850K
- Sales.com sold for $2.6M
- Target domain has category-defining premium
- Adjustment: +50-100% over closest comparable
- Range: $X - $Y based on comparables
```

**Method 2: Revenue Multiple**
```
If domain generates revenue:
Value = Annual Revenue × Multiple

Multiples by revenue type:
- Parking revenue: 2-4x
- Affiliate revenue: 3-5x
- Lead generation: 4-8x
- Subscription business: 5-10x

Example:
Domain generates $100K/year parking
Value = $100K × 3 = $300K
(Floor value; strategic value likely higher)
```

**Method 3: Traffic Multiple**
```
Value = Monthly Visitors × Value per Visitor

Value per visitor by traffic type:
- Generic type-in: $0.10 - $1.00
- Commercial intent: $1.00 - $10.00
- High-value category: $5.00 - $50.00

Example:
50,000 monthly type-ins × $2.00 = $100K
Capitalized at 3x = $300K minimum
```

**Method 4: Strategic Value Assessment**
```
Strategic Value = Cost of Alternative + Opportunity Value

Cost of Alternative:
- What would branding campaign cost without domain?
- SEO value to rank for keyword
- Brand confusion cost with alternative

Opportunity Value:
- Traffic acquisition savings
- Brand authority value
- Competitive advantage
```

### Domain Due Diligence

**Essential Checks**:
- [ ] **Ownership verification** — WHOIS, registrar records
- [ ] **Transfer restrictions** — Any liens, disputes, locks?
- [ ] **History** — Wayback Machine, prior use
- [ ] **Trademark conflicts** — USPTO, international marks
- [ ] **Backlink profile** — Quality, spam history
- [ ] **Traffic verification** — Analytics access
- [ ] **Revenue verification** — If income claimed
- [ ] **Penalty history** — Google Search Console
- [ ] **Registration history** — How long held, transfers

**Red Flags**:
| Issue | Risk |
|-------|------|
| Recent ownership changes | Possible flip or dispute |
| Trademark conflicts | Legal liability |
| Spam backlinks | SEO penalty risk |
| Prior adult/gambling use | Brand risk |
| UDRP history | Dispute vulnerability |
| Privacy WHOIS | Verify actual owner |

## IP Asset Valuation

### Types of IP Assets

| Asset Type | Valuation Approach |
|------------|-------------------|
| Patents | Income, cost, market approach |
| Trademarks | Relief from royalty, brand equity |
| Copyrights | Income stream, market comps |
| Trade secrets | Competitive advantage value |
| Software | Development cost, licensing revenue |

### IP Valuation Methods

**Income Approach**:
```
Value = Present Value of Future Income from IP

PV = Σ (Cash Flow_t / (1 + r)^t)

Where:
- Cash Flow = Revenue attributable to IP
- r = Discount rate (risk-adjusted)
- t = Time period
```

**Cost Approach**:
```
Value = Cost to Recreate IP

Components:
- Development costs
- Opportunity cost of time
- Failed attempts
- Inflation adjustment
```

**Market Approach**:
```
Value = Comparable Transaction × Adjustment

Find similar IP sales and adjust for:
- Size/scope differences
- Industry differences
- Economic conditions
- Strategic vs. financial buyer
```

### Relief from Royalty Method

```
Value = Revenue × Royalty Rate × PV Factor

Example (trademark):
- Revenue using trademark: $10M/year
- Industry royalty rate: 5%
- Implied royalty: $500K/year
- PV of 10 years at 15%: 5.02
- Value: $500K × 5.02 = $2.5M
```

## Contribution Valuation in Operating Agreements

### Assessing Contributed Asset Value

**Framework**:
1. Identify the asset being contributed
2. Determine appropriate valuation method
3. Assess stated value vs. market value
4. Identify valuation risks
5. Recommend adjustments if needed

**Key Questions**:
- How was the valuation determined?
- Is there an independent appraisal?
- What comparables support the value?
- Is the value realistic given market conditions?
- What are the tax implications of stated value?

### Example Analysis (Domain Contribution)

```
## Domain Contribution Analysis

**Asset**: 50% of Domain LLC (holding premium domain)
**Stated Value**: $500,000 (for 50% = $1M total domain value)
**Contributor**: Partner Entity

### Valuation Assessment

**Comparables**:
| Domain | Sale Price | Year | Relevance |
|--------|-----------|------|-----------|
| Shop.com | $3,500,000 | 2019 | Similar category |
| Sales.com | $2,600,000 | 2015 | Same industry term |
| Store.com | $850,000 | 2018 | Related vertical |

**Category Premium Analysis**:
- Domain is the defining term for a large industry
- Not just descriptive, but category-owning
- .com essential for credibility
- Premium over similar domains justified

**Valuation Range**:
- Floor (conservative): $1,500,000
- Mid-range: $3,000,000 - $5,000,000
- Ceiling (strategic buyer): $8,000,000+

**Assessment of Stated Value**:
Compare stated value to market evidence.

Possible explanations for variance:
1. Negotiated discount for partnership
2. Tax basis considerations
3. Historical acquisition cost
4. Incomplete market analysis

**Recommendation**:
Consider independent appraisal to document fair value for:
- Tax purposes (Section 721 contribution)
- 704(c) allocation tracking
- Future dispute prevention
```

## Severity Classification

### 🔴 RED — Valuation Issues

- Stated value >2x higher than market evidence
- No comparables support valuation
- Trademark conflicts exist
- Domain has penalty/spam history
- Ownership disputed or unclear

### 🟡 YELLOW — Review Needed

- Stated value differs from market by 30-50%
- Limited comparable data available
- Recent ownership changes
- Minor trademark concerns
- Traffic/revenue unverified

### 🟢 GREEN — Acceptable

- Value supported by comparables
- Clean ownership history
- No trademark conflicts
- Traffic/revenue verified (if claimed)
- Independent appraisal obtained

## Limitations

**This skill does NOT:**
- Provide certified appraisals
- Replace professional IP valuators
- Account for private sale dynamics
- Guarantee accuracy of estimates
- Consider all market conditions

**Always recommend:**
- Independent appraisal for high-value transactions
- Legal review of ownership and IP rights
- Verification of traffic and revenue claims
- Trademark clearance search
